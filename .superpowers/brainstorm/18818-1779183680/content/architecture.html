⚙️ 关键技术决策

路由策略
App Router + 路由组 (marketing) 隔离布局，博客用动态路由 [slug]
渲染策略
首页/功能/定价 → SSG 静态生成
博客列表 → ISR (revalidate: 3600)
博客详情 → generateStaticParams 预生成
样式方案
CSS Modules 组件级隔离
globals.css 定义 CSS 变量（颜色/间距）
渐变色系：#667eea → #764ba2
博客内容
MDX 文件存 content/blog/
gray-matter 解析 frontmatter
next-mdx-remote 渲染
部署
Vercel 一键部署
GitHub 仓库 → 自动 CI/CD
